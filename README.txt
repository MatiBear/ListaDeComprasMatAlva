-Gestionador de Productos/Lista de Compras-
Por Matias Alvarado

Descripcion:
Esta aplicacion incluye varios elementos que permite al usuario manejar productos que desean comprar de varias tiendas a la vez, personalizado la cantidad de los productos, y su presupuesto actual que pueden manejar en todo momento. Principalmente, esta la funcion de Inventario que almacena todos los productos que fueron comprados, y que el usuario puede manipular la cantidad o removerlos, para mantener un registro de estos articulos. Adicionalmente, pueden crear listas de los productos que desean escoger, siendo guardados para otras ocasiones donde deseen utilizar la lista para comprar directamente esos productos.

Requerimientos funcionales:
- Interfaz de presentacion del app antes de la pantalla principal.
- Poder comprar articulos de varias tiendas a la vez.
- Registrar, gestionar, y remover articulos de una lista.
- Tener un inventario del registro de elementos comprados, y poder manipularlos.
- Tener la capacidad de manipular el presupuesto.
- Tener puntos de interes (Tiendas) donde comprar objetos

Requerimientos No funcionales:
- Horarios de aviso para recordatorio de compras.
- Notificaciones de aviso cuando el inventario esta bajo.
- Ubicacion especificas de los lugares de interes por geolocalizacion.
- Poder buscar productos por escrito.
